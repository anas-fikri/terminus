# Roadmap & Saran Pengembangan Viewer/Editor

## 1. Fitur Prioritas Tinggi
- **Multi-file Tab/Pane**: Buka beberapa file sekaligus, tabbed atau split view.
- **Undo/Redo Edit Mode**: Dukungan undo/redo (Ctrl+Z/Ctrl+Y) saat edit.
- **Auto-Save & Recovery**: Simpan otomatis & pemulihan file jika app crash.
- **File History/Backup**: Simpan versi sebelumnya setiap kali save.
- **Drag & Drop File**: Bisa drag file ke viewer untuk dibuka.

## 2. UX & Navigasi
- **Breadcrumb Navigation**: Navigasi path file/folder di atas viewer.
- **Quick Open (Cmd+P)**: Cari & buka file cepat.
- **Keyboard Shortcuts Custom**: Pengaturan shortcut sesuai preferensi user.
- **Dark/Light Mode Toggle**: Ganti tema langsung dari UI.

## 3. Advanced Editing
- **Syntax Highlighting**: Deteksi & highlight syntax sesuai tipe file.
- **Line Numbering**: Tampilkan nomor baris di Edit Mode.
- **Code Folding**: Collapse/expand section (khusus code/markdown).
- **Linting/Validation**: Tampilkan error/warning saat edit file code.

## 4. Integrasi & Ekstensi
- **Plugin System**: Support plugin eksternal (misal: spellcheck, formatter).
- **Export/Import**: Export file ke format lain (PDF, HTML, dsb).
- **Cloud Sync**: Sinkronisasi file ke cloud (optional, bisa manual).

## 5. Kinerja & Stabilitas
- **Optimasi File Besar**: Pastikan viewer tetap responsif untuk file besar.
- **Profiling & Monitoring**: Tool untuk cek performa & resource usage.
- **Error Reporting**: Fitur kirim bug report langsung dari app.

## 6. Dokumentasi & Bantuan
- **User Guide**: Panduan penggunaan fitur-fitur utama.
- **FAQ & Troubleshooting**: Jawaban masalah umum.
- **Changelog**: Catatan perubahan tiap versi.

---

### Catatan
- Prioritas bisa disesuaikan kebutuhan user.
- Fokus awal: fitur yang langsung meningkatkan produktivitas & UX.
- Saran di atas bisa dikembangkan bertahap sesuai resource & feedback user.
